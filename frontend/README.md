Projeto técnico 

Sistema para inclusão de notas, curos e avaliações.

* Angular 7
* @ionic/angular 4.0


Requesitos:
------------

* Node 8.13+
* Npm 6+
* Ionic CLI 4.5

Instalação:
------------

```
$ npm install
$ ionic serve -l
```
