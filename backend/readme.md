Projeto técnico Backend

Sistema para inclusão de notas, curos e avaliações.

* Angular 7
* SPringBoot


Requesitos:
------------

* Java 8+
* Mysql 5+

Instalação:
------------

```
$ maven install
$ ionic serve -l
```
